# Git-it-ready

웹 기반 Git 시뮬레이터입니다. Git 명령어, 브랜치, 머지(merge), 원격 저장소 상호작용 등을 시각적으로 연습하고 이해할 수 있는 환경을 제공합니다.

## 주요 기능

- **터미널:** Git 명령어를 실행하여 저장소 상태를 조작할 수 있습니다.
- **시각화 그래프:** 커밋 히스토리, 브랜치, 원격 저장소 동기화 상태를 실시간으로 시각화합니다.
- **파일 에디터:** 시뮬레이터 환경 내에서 직접 파일을 생성하고 편집할 수 있습니다.
- **GitHub 인터페이스:** 풀 리퀘스트(PR)와 원격 저장소 워크플로우를 이해할 수 있습니다.
